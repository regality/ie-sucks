# Because IE Sucks

This is a collection of methods that IE fails to implement, which are
part of javascript. The functions implemented are:

    Array.every
    Array.filter
    Array.forEach
    Array.indexOf
    Array.lastIndexOf
    Array.map
    Array.reduce
    Array.reduceRight
    Array.some
    Function.bind
    Object.create
    Object.keys

To use, simply copy ie-sucks.js and add
`<script src="ie-sucks.js"></script>` to your wobsite.
